# 架空リフォーム会社HP
**地域に寄り添う、親しみやすいリフォーム会社**をテーマに、デザインから実装まで一人で完成させた作品です。
## デモ
 https://okoba-git.github.io/reform/index.html

## 制作期間
2026年1月 企画開始
2026年2月半ば デザイン完成、コーディング開始

## 使用技術
- HTML
- CSS
- JavaScript
- Figma（デザイン）
- AI（画像や文章の作成の補助として使用）

## 実装した機能
- ハンバーガーメニュー
- お知らせ一覧をJavaScriptで動的描画
- 問い合わせフォームの簡単なバリデーション機能
- レスポンシブ対応

## 工夫した点
- JavaScriptでお知らせデータを配列管理し、DOM操作で描画
- メディアクエリを最小限にし、`clamp()` と `flexbox` を活用してレスポンシブ対応
- 画像のalt属性を詳細に記入
- BEM/FLOCSSを参考に、保守性を意識したクラス命名

## 大変だった点
- デザインに関して知識がないのでテーマや方向性を決めるまでにかなり時間がかかりました。様々なサイトを参考にしながら親しみやすく安心できるデザインを目指しました。
- リフォームの流れページにて矢印をどう表示させるか悩み、疑似要素を使い後から挿入しました。
