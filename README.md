# 架空リフォーム会社HP
放課後や休日を利用して、**地域に寄り添う、親しみやすいリフォーム会社**をテーマに、デザインから実装まで一人で完成させた作品です。

[デモサイトはこちら](https://okoba-git.github.io/reform/index.html)

## 制作期間
- 2026年1月：当サイトの企画開始
- 2026年2月半ば：デザイン完成、コーディング開始
- 2026年3月：GitHub Pagesにて公開

## 使用技術
* HTML
* CSS
* JavaScript
* Figma（デザイン）
* chatGPT（画像や文章作成の補助として使用）

## 実装した機能
- ハンバーガーメニュー
- お知らせ一覧・施工事例一覧をJavaScriptで配列管理し、動的に描画
- フィルター機能
- 問い合わせフォームのバリデーション機能
- レスポンシブ対応
- CSSアニメーション
- スクロールアニメーション

## 工夫した点
- JavaScriptでお知らせデータを配列管理し、DOM操作で描画
- メディアクエリを最小限にし、`clamp()` と `flexbox` を活用
- 画像の`alt属性`を詳細に記入
- `BEM/FLOCSS`を参考に、保守性を意識したクラス命名

## 大変だった点
- デザインに関して知識がないのでテーマや方向性を決めるまでにかなり時間がかかりました。様々なサイトを参考にしながら親しみやすく安心できるデザインを目指しました。
- 雰囲気に合う素材を探しにもかなり時間がかかり、最終的にはchatGPTを活用し、素材の生成・選定を行いました。
- リフォームの流れページにて矢印をどう表示させるか悩み、疑似要素を使い後から挿入しました。
- スクロールアニメーションについて理解するまでに時間がかかり、何度も試行錯誤しながら実装しました。
